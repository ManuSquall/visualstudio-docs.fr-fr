# <a name="iactivescriptsitetraceinfo-interface"></a>IActiveScriptSiteTraceInfo, interface
Fournit une méthode pour envoyer des informations de traçage de script.  
  
## <a name="methods"></a>Méthodes  
  
|||  
|-|-|  
|Méthode|Description|  
|[IActiveScriptSiteTraceInfo::SendScriptTraceInfo Method](../../winscript/reference/iactivescriptsitetraceinfo-sendscripttraceinfo-method.md)|Envoie des informations de trace qui inclut le type d’événement, de contexte et de l’instruction de script.|
