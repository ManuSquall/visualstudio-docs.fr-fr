# <a name="iactivescriptstringcompare-interface"></a>IActiveScriptStringCompare, interface
Le `IActiveScriptCompare` interface est utilisée pour définir la méthode de comparaison de chaînes pour le moteur de script.  
  
## <a name="methods-in-vtable-order"></a>Méthodes dans l'ordre Vtable  
  
|Méthode|Description|  
|------------|-----------------|  
|[IActiveScriptStringCompare::StrComp](../../winscript/reference/iactivescriptstringcompare-strcomp.md)|Définit la méthode de comparaison de chaînes pour le moteur de script.|
