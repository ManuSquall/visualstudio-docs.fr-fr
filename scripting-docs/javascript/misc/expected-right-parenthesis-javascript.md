# <a name="expected-3939-javascript"></a>Attendu &#39;) &#39; (JavaScript)
Vous a tenté de placer une expression entre parenthèses, mais n’incluez pas la parenthèse fermante. Une expression doit être placée dans un jeu de parenthèses ouvrantes et fermantes. Notez l’utilisation de parenthèses dans l’exemple suivant.  
  
```JavaScript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajouter les parenthèses de droite à l’expression d’évaluation.
