# <a name="clear-method-weakmap-javascript"></a>clear, méthode (WeakMap) (JavaScript)
Supprime tous les éléments d'un objet `WeakMap`.  
  
## <a name="syntax"></a>Syntaxe  
  
```JavaScript  
weakmapObj.clear()  
```  
  
#### <a name="parameters"></a>Paramètres  
 `weakmapObj`  
 Obligatoire. Le `WeakMap` objet à effacer.
