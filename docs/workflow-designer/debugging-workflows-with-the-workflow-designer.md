# <a name="debugging-workflows-with-the-workflow-designer"></a>Débogage de workflows avec Workflow Designer

Le Concepteur de flux de travail Windows offre la possibilité de déboguer des workflows et des activités personnalisées. Le processus et le comportement sont semblables à celui du débogueur [!INCLUDE[vs2010](../misc/includes/vs2010_md.md)] par défaut. Les rubriques de cette section expliquent comment exécuter des procédures de débogage traditionnelles ainsi que le débogage de code XAML et d'expressions.
