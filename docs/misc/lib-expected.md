# &#39;Lib&#39; attendu
Une instruction `Declare` spécifie un nom de bibliothèque sans utiliser le mot clé `Lib`.  
  
 **ID d’erreur :** BC30218  
  
### Pour corriger cette erreur  
  
-   Ajoutez le mot clé `Lib` avant le nom de la bibliothèque.
