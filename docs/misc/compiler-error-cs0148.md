# Erreur du compilateur CS0148
Le délégué 'délégué' ne possède pas de constructeur valide  
  
 Vous avez importé et utilisé un programme managé \(qui utilise le common language runtime .NET Framework\) créé avec un autre compilateur. Ce compilateur a autorisé un constructeur [délégué](/dotnet/csharp/language-reference/keywords/delegate) incorrect. Pour plus d'informations, consultez [Délégués](/dotnet/csharp/programming-guide/delegates/index).
