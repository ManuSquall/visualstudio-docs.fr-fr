# La balise de commentaire XML &#39;returns&#39; n&#39;est pas autoris&#233;e pour une propri&#233;t&#233; &#39;WriteOnly&#39;
La balise de commentaire XML 'returns' n'est pas autorisée pour une propriété 'WriteOnly'. Le commentaire XML sera ignoré.  
  
 Un commentaire XML pour une déclaration de propriété en écriture seule ne peut pas contenir de balise \<returns\>.  
  
 **ID d’erreur :** BC42313  
  
### Pour corriger cette erreur  
  
-   Supprimez la balise \<returns\> non prise en charge.
