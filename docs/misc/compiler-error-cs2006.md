# Erreur du compilateur CS2006
Erreur de syntaxe de ligne de commande : 'texte' manquant pour l’option 'option'  
  
 La syntaxe de *option* nécessite du texte supplémentaire. Pour plus d’informations, consultez [Options du compilateur](/dotnet/csharp/language-reference/compiler-options/index).
