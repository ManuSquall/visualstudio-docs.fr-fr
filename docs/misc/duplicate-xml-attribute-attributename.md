# Attribut XML &#39;attributeName&#39; en double
Un attribut spécifié pour un littéral d’élément XML correspond à celui d’un attribut précédemment défini pour le littéral d’élément XML. Les noms d’attribut dans les littéraux d’élément XML doivent être uniques.  
  
 **ID d’erreur :** BC31149  
  
### Pour corriger cette erreur  
  
-   Renommez l’attribut avec un nom unique pour le littéral d’élément XML.
