# Erreur du compilateur CS0009
Impossible d’ouvrir le fichier de métadonnées 'file' — 'description'  
  
 Le fichier spécifié avec l’option du compilateur [\/reference](/dotnet/csharp/language-reference/compiler-options/reference-compiler-option) ne contient pas de métadonnées valides.
