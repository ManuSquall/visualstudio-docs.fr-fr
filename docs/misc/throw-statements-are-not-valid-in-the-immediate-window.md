# Les instructions &#39;Throw&#39; ne sont pas valides dans la fen&#234;tre Ex&#233;cution
Les instructions `Throw` ne peuvent être utilisées que dans le code source.  
  
 **ID d’erreur :** BC30725  
  
### Pour corriger cette erreur  
  
-   Supprimez les instructions `Throw` du code de débogage.
