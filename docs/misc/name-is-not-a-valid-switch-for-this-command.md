# &lt;name&gt; is not a valid switch for this command.
Cette erreur se produit généralement lorsqu'un commutateur ou une combinaison de commutateurs à un seul caractère ont été spécifiés avec une commande.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez dans la documentation la syntaxe à respecter pour la commande et recommencez.
