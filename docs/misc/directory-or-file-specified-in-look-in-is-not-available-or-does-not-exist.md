# Directory or file specified in Look In is not available or does not exist.
Cette erreur se produit généralement lorsque le chemin ou le nom de fichier spécifié dans la liste Parcourir est introuvable.  
  
### Pour corriger cette erreur  
  
1.  Entrez à nouveau le chemin ou le nom de fichier.  
  
     \- ou \-  
  
2.  Naviguez jusqu'au fichier ou dossier correct à l'aide de la liste Parcourir et de la zone de texte **Éléments disponibles**.
