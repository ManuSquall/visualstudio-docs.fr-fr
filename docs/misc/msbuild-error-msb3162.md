# MSBuild Error MSB3162
**MSB3162 : L'élément '\<package\>' n'a pas pu trouver l'élément dépendant '\<package\>'.**  
  
 Cette erreur se produit lorsqu'un package de programme d'amorçage dépend d'un autre package, mais le manifeste des composants requis du programme d'amorçage dépendant ne peut pas être trouvé.
