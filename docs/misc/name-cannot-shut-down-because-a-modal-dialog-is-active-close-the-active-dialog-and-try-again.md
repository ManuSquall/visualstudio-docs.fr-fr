# &lt;name&gt; cannot shut down because a modal dialog is active. Close the active dialog and try again.
Cette erreur se produit généralement lorsque le système d'exploitation tente de s'arrêter et qu'une boîte de dialogue est ouverte dans une application, ce qui empêche l'application de se fermer.  
  
### Pour corriger cette erreur  
  
1.  Fermez la boîte de dialogue modale, puis fermez l'application ou arrêtez le système d'exploitation.
