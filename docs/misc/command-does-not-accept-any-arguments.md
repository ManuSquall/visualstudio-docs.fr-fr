# Command does not accept any arguments.
Cette erreur se produit généralement lorsqu'un argument a été spécifié avec une commande qui n'accepte que des commutateurs.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez dans la documentation la syntaxe à respecter pour la commande et recommencez.
