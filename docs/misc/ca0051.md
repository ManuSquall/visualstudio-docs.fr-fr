# CA0051
Aucune règle n'a été sélectionnée.  
  
 Cette erreur indique que l'outil d'analyse de code managé est activé, mais aucune règle n'a été sélectionnée.  Pour corriger cette erreur, sélectionnez les règles appropriées.
