# discrete_distribution::probabilities
Retourne les probabilités stockées pour la distribution.  
  
## Syntaxe  
  
```  
std::vector<double> probabilities() const;  
```  
  
## Notes  
 Cette fonction membre retourne les valeurs des probabilités stockées.  
  
## Configuration requise  
 **En\-tête :** \<random\>  
  
 **Espace de noms :** std
