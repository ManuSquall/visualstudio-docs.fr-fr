# piecewise_linear_distribution::intervals
Retourne les intervalles stockés pour la distribution.  
  
## Syntaxe  
  
```  
std::vector<RealType> intervals() const;  
```  
  
## Notes  
 La fonction membre retourne les valeurs des intervalles stockés.  
  
## Configuration requise  
 **En\-tête :** \<random\>  
  
 **Espace de noms :** std
