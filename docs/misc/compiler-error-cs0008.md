# Erreur du compilateur CS0008
Erreur inattendue lors de la lecture des métadonnées dans le fichier 'fichier' — 'description'  
  
 Une DLL a été ouverte pour la récupération des métadonnées, mais elle est endommagée et les données n’ont pas pu être lues. Pour plus d'informations, consultez [\/reference \(Import Metadata\)](/dotnet/csharp/language-reference/compiler-options/reference-compiler-option).
