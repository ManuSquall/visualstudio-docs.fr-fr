# Erreur du compilateur CS0025
Impossible de trouver le fichier bibliothèque standard 'fichier'  
  
 Un fichier requis par le compilateur est introuvable. Vérifiez que le chemin est correct et que le fichier existe.  
  
 Si le fichier est un fichier système [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)], vous devrez peut\-être réparer votre installation [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] ou procéder à une réinstallation complète.
