# Command was not specified.
Cette erreur se produit généralement lorsqu'une chaîne vide a été entrée dans la fenêtre **Commande** ou la zone de texte **Rechercher\/Commande**.  
  
### Pour corriger cette erreur  
  
1.  Entrez une commande valide.  Pour plus d'informations, consultez [Commandes Visual Studio](../ide/reference/visual-studio-commands.md).
