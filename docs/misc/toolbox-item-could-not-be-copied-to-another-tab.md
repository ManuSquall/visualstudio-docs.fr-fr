# Toolbox item could not be copied to another tab.
Cette erreur se produit généralement quand l'élément est endommagé ou que la mémoire disponible sur l'ordinateur est insuffisante.  
  
### Pour corriger cette erreur  
  
1.  Fermez toutes les applications ouvertes, puis réessayez de déplacer l'élément.  
  
     \- ou \-  
  
2.  Sélectionnez un autre élément à déplacer.
