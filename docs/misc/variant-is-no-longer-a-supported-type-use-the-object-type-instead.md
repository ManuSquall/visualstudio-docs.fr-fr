# &#39;Variant&#39; n’est plus un type pris en charge&#160;; utilisez le type &#39;Object&#39; &#224; la place
Le type de données `Object` remplace `Variant`.  
  
 **ID d’erreur :** BC30804  
  
### Pour corriger cette erreur  
  
-   Déclarez l’élément comme `Object`.
