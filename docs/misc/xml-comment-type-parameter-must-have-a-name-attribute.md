# Le param&#232;tre de type de commentaire XML doit avoir un attribut &#39;name&#39;
Le paramètre de type de commentaire XML doit avoir un attribut 'name'. Le commentaire XML sera ignoré.  
  
 La balise de commentaire XML exige un attribut `name` qui est manquant ou incomplet.  
  
 **ID d’erreur :** BC42318  
  
### Pour corriger cette erreur  
  
-   Ajoutez l’attribut `name` obligatoire à la balise ou complétez\-le.
