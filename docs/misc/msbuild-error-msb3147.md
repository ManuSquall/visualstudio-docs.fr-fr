# MSBuild Error MSB3147
**MSB3147 : Impossible de trouver le fichier requis '\<fichier\>' dans '\<chemin d'accès\>'.**  
  
 Cette erreur se produit lorsque le programme d'amorçage ne trouve pas le fichier setup.bin sur le disque.
