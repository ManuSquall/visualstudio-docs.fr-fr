# Avertissement du compilateur (niveau&#160;1) CS2029
Valeur non valide pour '\/define' ; 'identificateur' n’est pas un identificateur valide  
  
 Cet avertissement se produit si la valeur utilisée dans l’option [\/define](/dotnet/csharp/language-reference/compiler-options/define-compiler-option) comporte des caractères non valides.  
  
 Vous ne pouvez pas supprimer cet avertissement avec l’option [\/nowarn](/dotnet/csharp/language-reference/compiler-options/nowarn-compiler-option).
