# CA0503
La propriété est déconseillée.  Utilisez la propriété de remplacement.  
  
 L'avertissement CA0503 est généré lorsqu'une propriété d'analyse du code déconseillée contient une valeur dans le fichier de configuration de build ou de projet.  Pour résoudre cet avertissement, utilisez la propriété spécifiée qui la remplace.
