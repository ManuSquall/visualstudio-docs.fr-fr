# Erreur du compilateur CS1555
Impossible de trouver 'classe' spécifié pour la méthode Main  
  
 Une classe a été spécifiée dans l’option du compilateur [\/main](/dotnet/csharp/language-reference/compiler-options/main-compiler-option), mais le nom de la classe est introuvable dans le code source.
