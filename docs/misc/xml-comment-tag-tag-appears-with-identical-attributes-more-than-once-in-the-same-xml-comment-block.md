# La balise de commentaire XML &#39;&lt;balise&gt;&#39; appara&#238;t plusieurs fois avec des attributs identiques dans le m&#234;me bloc de commentaire XML
La balise de commentaire XML \<balise\> apparaît plusieurs fois avec des attributs identiques dans le même bloc de commentaire XML. Le commentaire XML sera ignoré.  
  
 Le commentaire XML contient plusieurs balises avec des attributs identiques.  
  
 **ID d’erreur :** BC42305  
  
### Pour corriger cette erreur  
  
-   Supprimez la balise en double.
