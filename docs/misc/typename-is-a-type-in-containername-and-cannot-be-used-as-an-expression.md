# &#39;&lt;nom_type&gt;&#39; est un type dans &#39;&lt;nom_conteneur&gt;&#39; et ne peut pas &#234;tre utilis&#233; en tant qu’expression
L’un des noms utilisés est déjà déclaré comme nom d’un type et ne peut pas être utilisé dans une expression.  
  
 **ID d’erreur :** BC30691  
  
### Pour corriger cette erreur  
  
-   Attribuez un nom unique au membre.  
  
-   Mettez le nom du membre entre crochets.
