# Tab name &lt;name&gt; already exists.
Chaque onglet de la boîte à outils doit posséder un nom unique.  Cette erreur se produit généralement lorsque que le nom spécifié pour l'onglet de la boîte à outils est identique à celui de l'un des onglets existants.  
  
### Pour corriger cette erreur  
  
1.  Entrez un nom unique pour l'onglet.
