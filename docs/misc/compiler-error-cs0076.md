# Erreur du compilateur CS0076
Le nom d’énumérateur 'valeur\_\_' est réservé et ne peut pas être utilisé  
  
 Un élément dans une [énumération](/dotnet/csharp/language-reference/keywords/enum) ne peut pas avoir un identificateur appelé **valeur\_\_**.
