# MSBuild Error MSB3041
**Impossible de créer un nom de ressource de manifeste pour "'\<fichier\>'".**  
  
 La tâche `CreateManifestResourceName` ne peut pas créer de nom de ressource de manifeste pour le fichier spécifié.  
  
### Pour corriger cette erreur  
  
-   Pour plus d'informations sur la cause d'une erreur MSBuild et les résolutions possibles, consultez [Additional Resources](../msbuild/additional-msbuild-resources.md).
