# piecewise_linear_distribution::densities
Retourne les densités stockées pour la distribution.  
  
## Syntaxe  
  
```  
std::vector<double> densities() const;  
```  
  
## Notes  
 La fonction membre retourne les valeurs des densités stockées.  
  
## Configuration requise  
 **En\-tête :** \<random\>  
  
 **Espace de noms :** std
