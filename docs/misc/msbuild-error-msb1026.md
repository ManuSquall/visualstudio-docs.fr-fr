# MSBuild Error MSB1026
**Le fichier de schéma n'existe pas.**  
  
 Le fichier de schéma n'existe pas à l'emplacement spécifié.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez que le fichier de schéma existe dans le répertoire spécifié si vous avez indiqué l'emplacement absolu ou relatif du fichier de schéma.
