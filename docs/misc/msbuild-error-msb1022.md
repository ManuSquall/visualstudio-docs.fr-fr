# MSBuild Error MSB1022
**Le fichier réponse n'existe pas.**  
  
 Le fichier réponse n'existe pas à l'emplacement spécifié.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez que le fichier réponse existe dans le répertoire spécifié si vous avez indiqué l'emplacement absolu ou relatif du fichier réponse.
