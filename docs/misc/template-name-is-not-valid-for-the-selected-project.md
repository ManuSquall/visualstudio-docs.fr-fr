# Template &lt;name&gt; is not valid for the selected project.
Cette erreur se produit généralement lorsqu'un nom de modèle inconnu a été spécifié pour la commande de **Nouveau fichier** `File.NewFile` ou pour la commande **Ajouter un nouvel élément de solution** `File.AddNewSolutionItem`.  
  
### Pour corriger cette erreur  
  
1.  Entrez à nouveau la commande et ses arguments.  
  
     \- ou \-  
  
2.  Consultez les rubriques « Nouveau fichier, commande » ou « Ajouter un nouvel élément de solution, commande » pour obtenir la syntaxe correcte.
