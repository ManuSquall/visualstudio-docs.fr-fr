# Jeu de caract&#232;res [] vide ou [^] a &#233;t&#233; sp&#233;cifi&#233;.
Cette erreur se produit généralement quand l'option des expressions régulières a été sélectionnée et qu'une chaîne contenant une syntaxe incomplète a été spécifiée pour les modèles **Set of characters \[\]** ou **Character not in set \[^\]**.  Par exemple, `[}`.  
  
### Pour corriger cette erreur  
  
1.  Consultez la rubrique Expressions régulières pour obtenir des informations sur la syntaxe correcte du modèle, puis entrez à nouveau la chaîne.
