# La constante d’attribut &#39;&lt;nom_constante&gt;&#39; ne peut pas &#234;tre la cible d’une assignation
L’utilisateur a tenté d’assigner une valeur à une constante déclarée dans un attribut.  
  
 **ID d’erreur :** BC31510  
  
### Pour corriger cette erreur  
  
-   Supprimez l’assignation erronée.  
  
-   Si vous utilisez des attributs personnalisés que vous avez développés, redéfinissez le membre dans l’attribut en tant que champ.
