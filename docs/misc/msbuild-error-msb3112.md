# MSBuild Error MSB3112
**MSB3112 : Au moins deux assemblys ont la même identité '\<assembly\>'.**  
  
 Cet avertissement est généré lorsqu'au moins deux assemblys référencés ont la même identité.  
  
### Pour corriger cette erreur  
  
-   Supprimez l'un des assemblys référencés du projet.
