# File is not a valid .NET Framework or COM component
Vous essayez d'afficher dans l'Explorateur d'objets un fichier ne contenant pas d'objet valide.  Seuls [!INCLUDE[dnprdnshort](../code-quality/includes/dnprdnshort_md.md)] et les composants COM peuvent être affichés dans l'Explorateur d'objets.  
  
### Pour corriger cette erreur  
  
1.  Si vous avez sélectionné un fichier incorrect, sélectionnez le fichier approprié, puis réessayez.  
  
2.  Si, dans la boîte de dialogue **Open With**, vous avez sélectionné l'Explorateur d'objets par inadvertance, choisissez un autre programme pour ouvrir le fichier.
