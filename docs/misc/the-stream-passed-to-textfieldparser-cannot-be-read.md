# Le flux pass&#233; &#224; TextFieldParser est illisible
Le [TextFieldParser Object](/dotnet/visual-basic/language-reference/objects/textfieldparser-object) ne peut pas lire le flux qui lui a été passé. Cela peut provenir d’une tentative de lecture d’un fichier qui n’est pas un fichier texte.  
  
### Pour corriger cette erreur  
  
-   Vérifiez que le fichier est un fichier texte.
