# D&#233;pannage des exceptions&#160;: System.Data.SqlServerCe.SqlCeInvalidDatabaseFormatException
Une exception `System.Data.SqlServerCe.SqlCeInvalidDatabaseFormatException` se produit lorsque SQL Server Compact ouvre un fichier de base de données créé avec une version antérieure du produit.  
  
## Notes  
 Vous pouvez mettre à jour le fichier de base de données à l'aide de [!INCLUDE[vs_orcas_long](../debugger/includes/vs_orcas_long_md.md)] ou de l'API `System.Data.SqlServerCe.SqlCeEngine.Upgrade`.  
  
 Pour plus d’informations, consultez [SQL Server Compact 3.5 Class Library](http://go.microsoft.com/fwlink/?LinkID=102595).
