# Clipboard cannot be opened.
Cette erreur se produit généralement lorsque vous essayez de couper, copier ou coller un élément, mais que l'accès au Presse\-papiers est impossible.  
  
### Pour corriger cette erreur  
  
1.  Fermez les programmes susceptibles d'utiliser le Presse\-papiers et recommencez.
