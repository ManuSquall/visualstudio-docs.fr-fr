# D&#233;pannage des exceptions&#160;: System.Data.SqlServerCe.SqlCeTransactionInProgressException
Une exception `System.Data.SqlServerCe.SqlCeTransactionInProgressException` se produit lorsqu'une application essaie de modifier une base de données pendant qu'une autre transaction est en cours d'exécution. Pour plus d’informations, consultez [Bibliothèque de classes SQL Server Compact 3.5](http://go.microsoft.com/fwlink/?LinkID=102595).
