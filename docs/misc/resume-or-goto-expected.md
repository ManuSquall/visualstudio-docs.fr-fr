# &#39;Resume&#39; ou &#39;GoTo&#39; attendu
Une instruction `On Error` ne contient pas de clause `Resume` ou de clause `GoTo`. Elle doit utiliser l’une ou l’autre.  
  
 **ID d’erreur :** BC32019  
  
### Pour corriger cette erreur  
  
-   Ajoutez une clause `Resume` ou `GoTo` à l’instruction `On Error`.
