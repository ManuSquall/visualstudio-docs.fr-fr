# CA0069
Utilisation d'une autre plateforme.  *VersionFramework1* est introuvable.  Utilisation de *VersionFramework2* à la place.  Pour obtenir les meilleurs résultats d'analyse possibles, vérifiez que le .NET Framework approprié est installé.
