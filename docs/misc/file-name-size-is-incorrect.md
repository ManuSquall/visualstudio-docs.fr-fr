# File &lt;name&gt; size is incorrect.
Le fichier contient un nombre d'octets inférieur à celui qui est attendu, peut\-être parce que le fichier a été tronqué ou est endommagé.  
  
### Pour corriger cette erreur  
  
1.  Fermez Visual Studio, puis redémarrez\-le.  
  
2.  Si cette erreur persiste, réinstallez Visual Studio pour réparer le fichier.
