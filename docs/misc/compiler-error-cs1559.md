# Erreur du compilateur CS1559
Impossible d’utiliser 'object' pour la méthode Main, car il est importé  
  
 Une classe non valide a été spécifiée pour l’option du compilateur [\/main](/dotnet/csharp/language-reference/compiler-options/main-compiler-option) ; la classe ne peut pas être utilisée comme emplacement pour la méthode [Main](/dotnet/csharp/programming-guide/main-and-command-args/main-and-command-line-arguments).
