# An invalid file was specified for /resetsettings: &lt;filename&gt;.  /resetsettings will be ignored.
Cette erreur se produit généralement lorsque le fichier des paramètres n'est pas lisible, n'existe pas ou contient du code XML incorrect.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez que le nom de fichier est correct.  
  
2.  Vérifiez que le fichier est au format correct.  
  
3.  Vérifiez que le fichier contient du code XML correctement formé.
