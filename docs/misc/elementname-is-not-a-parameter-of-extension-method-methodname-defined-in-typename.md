# &#39;&lt;nom_&#233;l&#233;ment&gt;&#39; n’est pas un param&#232;tre de la m&#233;thode d’extension &#39;&lt;nom_m&#233;thode&gt;&#39; d&#233;fini dans &#39;&lt;nom_type&gt;&#39;
Un appel de procédure fournit un argument supplémentaire qui n’est pas défini par la méthode d’extension.  
  
 **ID d’erreur :** BC36585  
  
### Pour corriger cette erreur  
  
-   Supprimez l’argument superflu de la liste d’arguments.
