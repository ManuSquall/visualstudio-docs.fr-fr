# Erreur RC2159 du compilateur de ressources
Type d’accélérateur non valide \[ASCII ou VIRTKEY\]  
  
 Le champ `type` de l’instruction **ACCELERATORS** doit contenir la valeur ASCII ou VIRTKEY.
