# Option is not currently valid.
Cette erreur se produit généralement quand un argument ou un commutateur a été spécifié pour une commande alors qu'il n'est pas disponible.  Par exemple, si vous avez entré `>Edit.Find test /sel` alors qu'aucune sélection n'a été effectuée dans le fichier, le commutateur \/sel ne sera pas disponible.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez que les arguments et commutateurs entrés sont disponibles.  Pour plus d'informations, consultez [Commandes Visual Studio utilisant des arguments](../ide/reference/visual-studio-commands.md).
