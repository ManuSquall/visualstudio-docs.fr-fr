# Erreur du compilateur CS1043
{ ou ; attendu  
  
 Un accesseur de propriété a été déclaré de manière incorrecte. Pour plus d’informations, consultez [Utilisation de propriétés](/dotnet/csharp/programming-guide/classes-and-structs/using-properties).  
  
## Exemple  
 L’exemple suivant génère l’erreur CS1043.  
  
```  
// CS1043.cs // compile with: /target:library public class MyClass { public int DoSomething { get return 1;   // CS1043 set {} } // OK public int DoSomething2 { get { return 1;} } }  
```
