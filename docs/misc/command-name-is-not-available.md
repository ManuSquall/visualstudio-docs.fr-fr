# Command &lt;name&gt; is not available.
Cette erreur se produit généralement lorsqu'une commande est entrée dans la fenêtre **Commande** ou dans la zone **Rechercher\/Commande** et qu'elle n'est pas valide, selon l'état en cours de Visual Studio.  Par exemple, la commande `File.CloseSolution`  n'est pas valide si aucune solution n'est ouverte dans Visual Studio.  
  
### Pour corriger cette erreur  
  
1.  Choisissez une autre commande.  
  
     \- ou \-  
  
     Utilisez la commande avec une sélection différente.
