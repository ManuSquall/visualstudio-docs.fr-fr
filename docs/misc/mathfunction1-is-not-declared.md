# &#39;&lt;fonction_math&#233;matique1&gt;&#39; n’est pas d&#233;clar&#233;
'\<fonction\_mathématique1\>' n’est pas déclaré. Cette fonction a été déplacée dans la classe System.Math et s’appelle '\<fonction\_mathématique2\>'.  
  
 Plusieurs fonctions qui étaient intrinsèques à [!INCLUDE[vbprvb](../code-quality/includes/vbprvb_md.md)] dans les versions précédentes ont été déplacées vers l’espace de noms <xref:System.Math?displayProperty=fullName>. Leurs fonctionnalités sont ainsi plus généralement accessibles à tous les langages de programmation.  
  
 **ID d’erreur :** BC30819  
  
### Pour corriger cette erreur  
  
-   Utilisez les méthodes déclarées dans <xref:System.Math?displayProperty=fullName>.
