# Erreur RC2145 du compilateur de ressources 
ID DE LANGUE PRINCIPALE trop grand  
  
 L’ID DE LANGUE PRINCIPALE doit être un ID de langue hexadécimal valide. Consultez [Chaînes de langues et de pays\/régions](/visual-cpp/c-runtime-library/locale-names-languages-and-country-region-strings) dans la *Référence de la bibliothèque Runtime* pour obtenir la liste des ID de langue valides.
