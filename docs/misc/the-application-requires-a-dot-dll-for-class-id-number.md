# The application requires a .dll for class ID &lt;number&gt;.
Visual Studio n'a pas pu charger la DLL de la classe que vous avez spécifiée.  Il s'agit d'une erreur du système d'exploitation.  
  
### Pour corriger cette erreur  
  
1.  Réinstallez Visual Studio ou l'application concernée et recommencez.
