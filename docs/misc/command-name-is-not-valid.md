# Command &lt;name&gt; is not valid.
Cette erreur se produit généralement lorsqu'une commande ou l'alias d'une commande non reconnus par Visual Studio ont été entrés dans la zone de texte **Rechercher\/Commande** ou la fenêtre **Commande**.  Si un alias a été spécifié, l'intégralité de la commande apparaît à la place du nom d'alias dans le message d'erreur.  
  
### Pour corriger cette erreur  
  
1.  Entrez à nouveau le nom de la commande ou choisissez un nom de commande valide.
