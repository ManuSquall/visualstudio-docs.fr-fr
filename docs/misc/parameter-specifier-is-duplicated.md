# Le sp&#233;cificateur de param&#232;tre est en double
Un spécificateur de paramètre est appliqué plusieurs fois au paramètre  
  
 **ID d’erreur :** BC30785  
  
### Pour corriger cette erreur  
  
-   Supprimez les spécificateurs en double.
