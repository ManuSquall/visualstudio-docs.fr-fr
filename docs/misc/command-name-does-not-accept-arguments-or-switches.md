# Command &lt;name&gt; does not accept arguments or switches.
Cette erreur se produit généralement lorsqu'un argument ou un commutateur ont été entrés pour une commande ou un alias qui n'acceptent pas d'arguments ou de commutateurs.  Si un alias a été spécifié, l'intégralité du nom de la commande apparaît dans le texte du message à la place du nom d'alias.  Pour obtenir la liste de commandes valides qui utilisent des commutateurs et des arguments, consultez [Commandes Visual Studio](../ide/reference/visual-studio-commands.md).  
  
### Pour corriger cette erreur  
  
1.  Entrez à nouveau la commande et les arguments, ou entrez une nouvelle commande.
