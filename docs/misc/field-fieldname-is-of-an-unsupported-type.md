# Le champ &#39;&lt;nom_champ&gt;&#39; est d’un type non pris en charge.
Un champ est déclaré comme un type qui ne peut pas être représenté par [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].  
  
 **ID d’erreur :** BC30656  
  
### Pour corriger cette erreur  
  
1.  Attribuez au champ un type pris en charge.  
  
2.  Développez un nouveau type décrivant les données du champ.
