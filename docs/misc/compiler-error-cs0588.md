# Erreur du compilateur CS0588
Erreur interne du compilateur : étape 'LEX'  
  
 Essayez de déterminer si le compilateur échoue en raison de son incapacité à analyser une syntaxe inattendue. Si ce n’est pas le cas, essayez d’[Nous contacter](../ide/talk-to-us.md).
