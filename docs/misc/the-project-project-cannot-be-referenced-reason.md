# The project &#39;project&#39; cannot be referenced. &lt;reason&gt;
Une référence entre projets à un projet donné n'a pas pu être effectuée.  
  
 La raison la plus courante de cette erreur est une référence à un projet qui génère un fichier .exe.  Cette erreur indique également qu'un projet fait référence à un projet dont la sortie ne correspond pas à un assembly \(par exemple une application C\+\+ native\).
