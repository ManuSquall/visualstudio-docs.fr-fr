# The &lt;name&gt; switch requires a value.
Cette erreur se produit généralement lorsqu'un commutateur nécessitant une valeur est spécifié sans valeur.  
  
### Pour corriger cette erreur  
  
1.  Entrez à nouveau la commande et entrez une valeur pour le commutateur.  
  
     \- ou \-  
  
     Entrez à nouveau la commande et ne spécifiez pas de valeur pour le commutateur.
