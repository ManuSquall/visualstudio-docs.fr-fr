# This tool could not be executed. Please ensure that it points to a valid executable.
Cette erreur se produit généralement lorsque vous tentez de démarrer un outil externe et que l'exécution de cet outil échoue.  Cette erreur peut se produire parce que le fichier spécifié n'est pas un fichier de commandes ou un fichier exécutable valide.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez que l'outil externe est un fichier de commandes ou un fichier exécutable valide à l'aide de la boîte de dialogue **Outils externes** du menu **Outils**.
