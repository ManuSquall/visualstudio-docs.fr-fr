# poisson_distribution::mean
Retourne le paramètre de distribution mean.  
  
## Syntaxe  
  
```  
RealType mean() const;  
```  
  
## Notes  
 La fonction membre retourne la valeur stockée `mean`.  
  
## Configuration requise  
 **En\-tête :** \<random\>  
  
 **Espace de noms :** std
