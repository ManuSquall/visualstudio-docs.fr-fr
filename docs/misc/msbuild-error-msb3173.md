# MSBuild Error MSB3173
**MSB3173 : Impossible d'écrire le manifeste '\<fichier\>'. '  \<erreur\>'**  
  
 Cette erreur est générée lorsque le processus de génération rencontre un problème général lors de l'écriture d'un fichier manifeste.  
  
### Pour corriger cette erreur  
  
-   Pour plus d'informations sur la cause d'une erreur MSBuild et les résolutions possibles, consultez [Additional Resources](../msbuild/additional-msbuild-resources.md).
