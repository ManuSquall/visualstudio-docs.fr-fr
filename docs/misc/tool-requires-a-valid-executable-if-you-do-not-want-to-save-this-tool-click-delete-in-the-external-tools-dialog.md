# L’outil n&#233;cessite un ex&#233;cutable valide. Si vous ne voulez pas enregistrer cet outil, cliquez sur Supprimer dans la bo&#238;te de dialogue Outils externes.
Cette erreur se produit généralement quand un outil a été ajouté à la boîte de dialogue Outils externes mais qu'aucun exécutable n'a été spécifié.  
  
### Pour corriger cette erreur  
  
1.  Si vous avez l'intention d'enregistrer l'outil, cliquez sur **OK** et entrez un nom de fichier dans la zone **Commande** de l'outil ou choisissez le bouton de sélection \(...\) pour rechercher un fichier.  
  
2.  Si vous n'avez pas l'intention d'enregistrer l'outil, cliquez sur **OK**, puis sur **Supprimer** dans la boîte de dialogue **Outils externes**.
