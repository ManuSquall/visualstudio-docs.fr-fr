# MSBuild Error MSB1021
**Impossible de créer une instance du journal.**  
  
 Une instance du journal ne peut pas être créée à partir de son assembly.  
  
### Pour corriger cette erreur  
  
-   Pour plus d'informations sur la cause d'une erreur MSBuild et les résolutions possibles, consultez [Additional Resources](../msbuild/additional-msbuild-resources.md).
