# CA0501
Impossible de lire le rapport de sortie  
  
 Cet avertissement se produit lorsque le rapport de sortie ne peut pas être écrit pour l'une des raisons suivantes :  
  
-   Disque plein  
  
-   Pas d'autorisations d'écriture dans le répertoire  
  
-   Le répertoire n'existe pas
