# Erreur du compilateur CS0586
Erreur interne du compilateur : étape 'stage'  
  
 Essayez de déterminer si le compilateur échoue en raison de son incapacité à analyser une syntaxe inattendue. Si ce n’est pas le cas, essayez [Nous contacter](../ide/talk-to-us.md).
