# CA0504
Le répertoire de règles a été ignoré car il n'existe pas  
  
 L'avertissement CA504 est généré lorsqu'un répertoire spécifié dans la propriété **CodeAnalysisRuleDirectories** d'un fichier de projet ou de configuration de build est introuvable.  Pour résoudre ce problème, entrez un chemin d'accès au répertoire qui soit valide.
