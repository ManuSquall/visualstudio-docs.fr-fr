# L’espace ne peut pas s’afficher ici
Le nom d’un littéral d’élément XML comprend un ou plusieurs espaces.  
  
 **ID d’erreur :** BC31177  
  
### Pour corriger cette erreur  
  
-   Supprimez les espaces du nom du littéral d’élément XML.
