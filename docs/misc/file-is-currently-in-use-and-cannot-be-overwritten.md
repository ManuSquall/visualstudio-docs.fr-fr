# File is currently in use and cannot be overwritten.
Le fichier en cours ne peut pas être enregistré.  Cette erreur se produit généralement lorsque vous essayez d'enregistrer un fichier sous le même nom qu'un fichier ouvert.  
  
### Pour corriger cette erreur  
  
1.  Fermez le fichier qui possède le même nom et recommencez.  
  
     \- ou \-  
  
     Renommez le fichier et enregistrez\-le ensuite.
