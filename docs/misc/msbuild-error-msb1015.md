# MSBuild Error MSB1015
**MSBuild ne peut pas être exécuté sur cette version du système d'exploitation.  Il n'est pris en charge que sur Windows 2000, Windows XP et versions ultérieures.**  
  
 [!INCLUDE[vstecmsbuild](../extensibility/internals/includes/vstecmsbuild_md.md)] n'est pas pris en charge sur cette version du système d'exploitation.  
  
### Pour corriger cette erreur  
  
1.  Installez et exécutez [!INCLUDE[vstecmsbuild](../extensibility/internals/includes/vstecmsbuild_md.md)] sur l'une des versions prises en charge du système d'exploitation.
