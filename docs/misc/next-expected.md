# &#39;Next&#39; attendu
Une construction `On Error Resume` se produit sans `Next` final.  
  
 **ID d’erreur :** BC30003  
  
### Pour corriger cette erreur  
  
-   Ajoutez le mot clé `Next` à la fin de l’instruction `On Error`.
