# &#39;&lt;nom&gt;&#39; est d&#233;clar&#233; dans le projet &#39;&lt;nom_projet1&gt;&#39;, qui n’est pas r&#233;f&#233;renc&#233; par le projet &#39;&lt;nom_projet2&gt;&#39;
Un élément de programmation est défini dans un projet auquel le projet actuel ne fait pas référence.  
  
 **ID d’erreur :** BC32004  
  
### Pour corriger cette erreur  
  
-   Dans le projet actif, ajoutez une référence au projet qui contient l’élément souhaité.
