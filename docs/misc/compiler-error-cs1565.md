# Erreur du compilateur CS1565
Options spécifiées en conflit : fichier de ressources Win32 ; icône Win32  
  
 Il n’est pas permis de spécifier l’option de compilateur [\/win32res](/dotnet/csharp/language-reference/compiler-options/win32res-compiler-option) et l’option de compilateur [\/win32ico](/dotnet/csharp/language-reference/compiler-options/win32icon-compiler-option) dans une même compilation.
