# Erreur irr&#233;cup&#233;rable RC1207 du compilateur de ressources 
page de codes par défaut non valide  
  
 La page de codes spécifiée par la variable d’environnement **RCCODEPAGE** n’est pas valide. Pour plus d’informations, consultez [Code Pages](/visual-cpp/c-runtime-library/code-pages) dans *Run\-Time Library Reference*.
