# Project type is not valid.
Cette erreur se produit généralement lorsqu'une valeur incorrecte a été spécifiée pour l'argument *templatename* de la commande **Ajouter un nouveau projet**.  
  
### Pour corriger cette erreur  
  
1.  Vérifiez dans la documentation la syntaxe à respecter pour la commande **Ajouter un nouveau projet**, puis réessayez.
