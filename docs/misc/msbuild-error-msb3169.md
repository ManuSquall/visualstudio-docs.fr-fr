# MSBuild Error MSB3169
**MSB3169 : Une erreur s'est produite lors de la génération d'un programme d'amorçage : '\<erreur\>'**  
  
 Ce message affiche les erreurs qui ne sont pas interceptés par la génération du programme d'amorçage.  
  
### Pour corriger cette erreur  
  
-   Pour plus d'informations sur la cause d'une erreur MSBuild et les résolutions possibles, consultez [Additional Resources](../msbuild/additional-msbuild-resources.md).
