# MSBuild Error MSB3160
**MSB3160 : Avertissement de validation Xml dans le fichier '\<fichier\>' : '\<erreur\>'**  
  
 Cet avertissement est généré lorsque le produit ou le fichier package.xml n'est pas conforme au schéma XML.  La dernière partie de cet avertissement est un message d'avertissement spécifique.  
  
### Pour corriger cette erreur  
  
-   Mettez à jour le fichier XML pour résoudre l'erreur de validation.
