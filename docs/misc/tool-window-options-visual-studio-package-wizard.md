# options de la fen&#234;tre Outil, Assistant Package Visual Studio
Cette étape vous permet de définir le nom de la fenêtre et l’ID de commande de votre fenêtre Outil.  
  
 **Nom de fenêtre**  
 Permet d’entrer le nom qui sera affiché dans la barre de titre de la fenêtre Outil. Ce nom sera également celui de la commande de menu qui ouvre la fenêtre Outil.  
  
 La commande de menu de votre fenêtre Outil est ajoutée au sous\-menu **Autres fenêtres** du menu **Affichage**.  
  
 **ID de commande**  
 Permet de définir l’ID de commande de votre fenêtre Outil. Cette constante est utilisée dans le code généré pour faire référence à cet élément de menu.  
  
 **Annuler**  
 Annule la création du VSPackage.  
  
 **Précédent**  
 Revient à la page précédente de l’Assistant.  
  
 **Terminer**  
 Crée le VSPackage.
