# <a name="-devenvexe"></a>/? (devenv.exe)

Affiche une boîte de message présentant tous les commutateurs `devenv`, avec une brève description de chacun d’eux.

## <a name="syntax"></a>Syntaxe

```shell
devenv /?
```
