# <a name="useenv-devenvexe"></a>/UseEnv (devenv.exe)
Démarre [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] et charge des variables d’environnement dans la boîte de dialogue **Répertoires VC++**.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
Devenv /useenv  
```  
  
## <a name="example"></a>Exemple  
 L’exemple suivant démarre [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] et charge des variables d’environnement dans la boîte de dialogue **Répertoires VC++**.  
  
```  
Devenv.exe /useenv  
```
