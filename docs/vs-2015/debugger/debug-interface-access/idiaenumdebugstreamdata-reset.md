# <a name="idiaenumdebugstreamdatareset"></a>IDiaEnumDebugStreamData::Reset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Vous trouverez la dernière version de cette rubrique dans [IDiaEnumDebugStreamData::Reset](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiaenumdebugstreamdata-reset).  
  
Réinitialise au début d’une séquence énumérée.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp#  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>Valeur de retour  
 Retourne `S_OK`.
