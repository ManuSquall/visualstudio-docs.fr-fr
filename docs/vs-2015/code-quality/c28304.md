# <a name="c28304"></a>C28304
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Vous trouverez la dernière version de cette rubrique dans [C28304](https://docs.microsoft.com/visualstudio/code-quality/c28304).  
  
avertissement C28304 : paramètre de référence pour C++ < nom_paramètre >, un placé `_Notref_` opérateur a été appliqué à \<jeton >.  
  
 Le `_Notref_` opérateur doit être utilisé uniquement dans des circonstances particulières impliquant des paramètres de référence C++ et uniquement dans les macros fournie par le système. Il doit être immédiatement suivi par un `_Deref_` opérateur ou une annotation fonctionnelle.
