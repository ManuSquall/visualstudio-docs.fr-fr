# <a name="c28020"></a>C28020
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avertissement C28020 : l’expression \<expr > n’est pas vrai dans cet appel  
  
 Cet avertissement est émis lorsque le \_Satisfies\_ expression répertoriée n’est pas vrai. Fréquemment, cela indique un paramètre incorrect.  
  
 Si cela se produit sur une déclaration de fonction, les annotations indiquent une condition impossible.
