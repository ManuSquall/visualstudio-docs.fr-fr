# <a name="c28214"></a>C28214
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

AVERTISSEMENT C28214 : les noms des paramètres d’attribut doivent être P1... P9  
  
 Cet avertissement indique que lorsque vous construisez une déclaration d’annotation, les noms de paramètres sont limités à P1... P9. Cet avertissement ne doit jamais se produire normalement.
