# <a name="c28217"></a>C28217
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Vous trouverez la dernière version de cette rubrique dans [C28217](https://docs.microsoft.com/visualstudio/code-quality/c28217).  
  
avertissement C28217 : pour la fonction, le nombre de paramètres pour l’annotation ne correspond pas trouvé dans le fichier  
  
 Les annotations sur la ligne actuelle et sur la ligne dans le message ne sont pas cohérents. Cela ne doit pas être possible si les macros standards sont utilisées pour les annotations ; Cet avertissement n’est pas prévu pour être affichés dans une utilisation normale.
