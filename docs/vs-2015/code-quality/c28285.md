# <a name="c28285"></a>C28285
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avertissement C28285 : pour la fonction, erreur de syntaxe dans le paramètre  
  
 L’outil d’analyse de Code signale cet avertissement lorsqu’une erreur est rencontrée dans le fichier de modèle. Certaines erreurs de fichier source peuvent également entraîner des erreurs de ce type.
