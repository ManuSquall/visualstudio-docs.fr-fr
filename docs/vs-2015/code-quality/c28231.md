# <a name="c28231"></a>C28231
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avertissement C28231 : L'annotation n'est valide que sur un tableau  
  
 Cet avertissement indique qu’un argument à une annotation doit être un tableau, et un autre type a été rencontrée.
