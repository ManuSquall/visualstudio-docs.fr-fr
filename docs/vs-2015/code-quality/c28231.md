# <a name="c28231"></a>C28231
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

AVERTISSEMENT C28231 : l’annotation n’est valide que sur un tableau  
  
 Cet avertissement indique qu’un argument d’une annotation doit être un tableau et qu’un autre type a été rencontré.
