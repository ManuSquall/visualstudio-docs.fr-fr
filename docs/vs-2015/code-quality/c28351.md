# <a name="c28351"></a>C28351
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Vous trouverez la dernière version de cette rubrique dans [C28351](https://docs.microsoft.com/visualstudio/code-quality/c28351).  
  
avertissement C28351 : l’annotation \<annotation > décrit où une valeur dynamique (une variable) ne peut pas être utilisée dans la condition.  
  
 Cet avertissement est émis lorsqu’une annotation est appliquée, où le type C/C++ est inspecté.  
  
 L’expression dans le `_When_` doit être évaluée à une constante. Le `_When_` est ignoré.
