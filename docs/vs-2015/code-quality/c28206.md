# <a name="c28206"></a>C28206
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

AVERTISSEMENT C28206 : \<expression> : l’opérande gauche pointe vers un struct, utilisez `->`  
  
 Cet avertissement est signalé lorsque la notation de déréférencement du pointeur de struct `->` était attendue.
