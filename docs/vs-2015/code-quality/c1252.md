Vous trouverez la dernière version de cette rubrique dans [C1252](https://docs.microsoft.com/visualstudio/code-quality/c1252).  
  
avertissement C1252 : dépendance circulaire ou manquante entre les plug-ins : GUID requis  
  
 L’outil d’analyse de Code signale cet avertissement lorsqu’il existe une erreur interne avec des dépendances de plug-in, pas dans le code en cours d’analyse.
