# <a name="c28350"></a>C28350
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avertissement C28350 : l’annotation \<annotation > décrit une situation qui n’est pas applicable de manière conditionnelle.  
  
 Généralement, cet avertissement est généré lorsqu’une annotation est appliquée, où le type C/C++ est inspecté.
