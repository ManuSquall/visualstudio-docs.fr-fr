# <a name="xml-documentation-comments-javascript"></a>Commentaires de documentation XML (JavaScript)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

En JavaScript, vous pouvez créer de la documentation pour votre code en incluant des étiquettes XML dans des champs de commentaires spéciaux dans le code source. Ces balises XML sont utilisées par IntelliSense pour fournir des informations sur les objets JavaScript.  
  
 L'emplacement des commentaires de code varie selon la balise.  
  
 Les balises suivantes sont prises en charge :  
  
- [\<deprecated>](../ide/deprecated-javascript.md)  
  
- [\<field>](../ide/field-javascript.md)  
  
- [\<loc>](../ide/loc-javascript.md)  
  
- [\<param>](../ide/param-javascript.md)  
  
- [\<returns>](../ide/returns-javascript.md)  
  
- [\<signature>](../ide/signature-javascript.md)  
  
- [\<summary>](../ide/summary-javascript.md)  
  
- [\<value>](../ide/value-javascript.md)  
  
- [\<var>](../ide/var-javascript.md)
