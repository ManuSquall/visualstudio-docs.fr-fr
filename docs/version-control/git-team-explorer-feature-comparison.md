# <a name="side-by-side-comparison-of-git-and-team-explorer"></a>Comparaison côte à côte de git et Team Explorer

Nous avons lancé la première version d’une nouvelle expérience git dans Visual Studio 2019 [version 16,8](/visualstudio/releases/2019/release-notes/). Cette nouvelle expérience permet de réduire le basculement de contexte avec une simple fenêtre de **modifications git** qui comprend des tâches git courantes. Il comprend également une fenêtre de **référentiel git** à l’écran pour les opérations git plus avancées, telles que la gestion des branches et l’exploration des référentiels.

Si vous avez utilisé Team Explorer, voici un guide pas à pas qui explique comment vous pouvez utiliser la nouvelle expérience git.

> [!NOTE]
> La section suivante contient des captures d’écran dimensionnées pour tenir dans les colonnes d’une table. Cliquez sur chaque capture d’écran pour en afficher une version plus volumineuse. (Si vous utilisez un appareil tactile, appuyez sur chaque capture d’écran pour en afficher une version plus importante.)
