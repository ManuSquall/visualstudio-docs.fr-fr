# <a name="c28212"></a>C28212
avertissement C28212 : expression de pointeur attendue pour l’annotation

 Cet avertissement indique que le paramètre numéroté à une annotation (pas la fonction annotée) est censé être un type pointeur ou tableau, mais un autre type a été rencontrée. L’annotation doit être corrigé.
