# C28221
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28221 : Expression de chaîne attendue pour le paramètre dans l'annotation  
  
 Cet avertissement indique qu'un paramètre d'une annotation devrait être une chaîne, et un autre type a été rencontré.  Cela indique généralement une macro d'annotation incorrectement codée et qui n'est pas vue en l'utilisation normale.
