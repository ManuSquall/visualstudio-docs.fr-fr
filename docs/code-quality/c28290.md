# C28290
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28290 : L'annotation pour la fonction contient plus de Externals que le nombre réel de paramètres  
  
 L'outil d'Analyse du Code inscrit cet avertissement lorsque l'annotation pour la fonction contient plus de Externals que le nombre réel de paramètres.
