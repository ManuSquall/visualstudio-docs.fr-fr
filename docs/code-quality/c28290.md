# <a name="c28290"></a>C28290
avertissement C28290 : l’annotation pour la fonction contient plus d’Externals que le nombre réel de paramètres  
  
 L’outil d’analyse du Code signale cet avertissement lorsque l’annotation pour la fonction contient plus d’Externals que le nombre réel de paramètres.
