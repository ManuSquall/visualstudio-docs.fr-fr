# <a name="c26497-useconstexprforfunction"></a>C26497 USE_CONSTEXPR_FOR_FUNCTION

Cet fonction % de fonction peut être marqué `constexpr` si l’évaluation au moment de la compilation est souhaitée. Consultez [C++ Core Guidelines F.4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rf-constexpr).
