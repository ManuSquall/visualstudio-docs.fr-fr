# <a name="c1250"></a>C1250
avertissement C1250 : Impossible de charger le plug-in.  
  
 L’outil d’analyse du Code signale cet avertissement lorsqu’il existe une erreur interne du plug-in, et non dans le code en cours d’analyse.
