# C1256
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Avertissement C1256 : PCH doit être accessible en lecture et écriture.  
  
 L'outil d'analyse du code inscrit cet avertissement en cas d'erreur interne dans l'outil, pas dans le code qui est analysé.
