# <a name="ca0504"></a>CA0504
Le répertoire de règles a été ignoré car il n’existe pas  
  
 Avertissement CA504 est généré lorsqu’un répertoire spécifié dans le **CodeAnalysisRuleDirectories** Impossible de trouver la propriété dans un fichier de configuration de projet ou de la build. Pour résoudre ce problème, entrez un chemin d’accès de répertoire valide.
