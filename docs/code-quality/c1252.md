# C1252
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C1252: Dépendance circulaire ou manquante entre les plug\-ins : GUID requit.  
  
 L'outil d'analyse du code signale cet avertissement en cas d'erreur interne avec les dépendences du plug\-in, et non dans le code qui est analysé.
