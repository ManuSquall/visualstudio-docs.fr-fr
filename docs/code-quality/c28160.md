# C28160
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28160 : Annotation d'erreur  
  
 Cet avertissement est signaler lorsqu'une annotation `__drv_error` a été rencontrée.  Cette annotation est utilisée pour signaler les pratiques de codage qui doivent être résolues, et peut être utilisée avec une annotation `__drv_when` pour indiquer des combinaisons spécifiques des paramètres.
