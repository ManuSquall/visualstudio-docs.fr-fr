# <a name="c26498-use_constexpr_for_functioncall"></a>C26498 USE_CONSTEXPR_FOR_FUNCTIONCALL

Cet appel de fonction% function% peut utiliser `constexpr` si l’évaluation au moment de la compilation est souhaitée. Consultez [C++ Core instructions con.5](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-constexpr).
