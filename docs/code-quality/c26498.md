# <a name="c26498-useconstexprforfunctioncall"></a>C26498 USE_CONSTEXPR_FOR_FUNCTIONCALL

Cet fonction fonction appel % peut utiliser `constexpr` si l’évaluation au moment de la compilation est souhaitée. Consultez [C++ Core instructions con.5](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-constexpr).
