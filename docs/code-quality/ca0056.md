# <a name="ca0056"></a>CA0056
Un fichier de projet a une version incorrecte de l’outil d’analyse.

 Cet avertissement se produit si FxCopCmd.exe est appelé manuellement avec un fichier de projet qui a une version de l’outil analyse du code incorrect.
