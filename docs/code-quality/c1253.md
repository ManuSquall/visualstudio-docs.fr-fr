# <a name="c1253"></a>C1253
avertissement C1253 : Impossible de charger le fichier de modèle.  
  
 L’outil d’analyse du Code signale cet avertissement lorsqu’il existe une erreur interne dans le fichier de modèle, et non dans le code en cours d’analyse.
