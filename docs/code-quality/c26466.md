# <a name="c26466-nostaticdowncastpolymorphic"></a>C26466 NO_STATIC_DOWNCAST_POLYMORPHIC
  N’utilisez pas `static_cast` downcasts. Un cast d’un type polymorphe doit utiliser dynamic_cast. Consultez [Type.2 des recommandations C++ Core](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Pro-type-downcast).
