# <a name="c26463-useconstforelements"></a>C26463 USE_CONST_FOR_ELEMENTS
  Les éléments du tableau '% tableau %' sont assignés qu’une seule fois, marquez les éléments `const`. Consultez [C++ Core instructions con.4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction).
