# C28213
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28213 : L'annotation `_Use_decl_annotations_` doit être utilisée pour référencer, sans modification, une déclaration antérieure. \<pourquoi\>  
  
 Cet avertissement est stocké lorsqu'une déclaration précédente est référencée sans l'annotation requise `_Use_decl_annotations_`.
