# <a name="c28213"></a>C28213
avertissement C28213 : le `_Use_decl_annotations_` annotation doit être utilisée pour référencer, sans modification, une déclaration antérieure. \<Pourquoi >  
  
 Cet avertissement est émis lorsqu’une déclaration antérieure est référencée sans requis `_Use_decl_annotations_` annotation.
