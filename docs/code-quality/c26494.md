# <a name="c26494-varusebeforeinit"></a>C26494 VAR_USE_BEFORE_INIT

La variable « %variable% » n’est pas initialisée. Initialisez toujours un objet. Consultez [Type.5 des recommandations C++ Core](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#SS-type).
