# <a name="c28301"></a>C28301
avertissement C28301 : Aucune annotation pour la première déclaration de \<fonction >.\< Note1 > voir \<filename > (\<ligne >). \<Note2 >

 Cet avertissement est émis lorsque les annotations sont introuvables dans la première déclaration d’une fonction donnée.
