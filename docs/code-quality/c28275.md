# <a name="c28275"></a>C28275
avertissement C28275 : le paramètre _Macro_value\_ a la valeur null

 Cet avertissement indique qu’il existe une erreur interne dans le fichier de modèle, et non dans le code en cours d’analyse. Le *macroValue* fonction a été appelée sans paramètre.
