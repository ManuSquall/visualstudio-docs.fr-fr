# <a name="c28246"></a>C28246
avertissement C28246 : l’annotation pour la fonction '\<nom >'-paramètre '\<paramètre >' ne correspond pas le type du paramètre  
  
 Un opérateur __deref a été appliqué à un type non pointeur lors de la création d’une annotation.
