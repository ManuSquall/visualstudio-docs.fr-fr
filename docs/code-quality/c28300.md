# C28300
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28300 : \<parameter\_name\>: Opérandes d'expression des types incompatibles pour l'opérateur \<operator\_name\>  
  
 Cet avertissement est émis lorsque les types d'opérandes dans un paramètre ne sont pas compatibles avec l'opérateur.
