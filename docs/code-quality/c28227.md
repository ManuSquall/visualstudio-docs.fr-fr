# C28227
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28227 : L'annotation ne peut pas également être un PrimOp \(voir la déclaration antérieure\)  
  
 Cet avertissement indique que l'annotation nommée est déclarée comme annotation ordinaire, et également précédemment déclaré comme PrimOp.  Cela ne doit pas être possible si les macros standard sont utilisées pour les annotations ; cet avertissement n'est pas destiné à être vu dans l'utilisation classique.
