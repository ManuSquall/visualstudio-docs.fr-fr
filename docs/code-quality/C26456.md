# <a name="c26456-donthideoperators"></a>C26456 DONT_HIDE_OPERATORS

Masquage des méthodes de base qui ne sont pas virtuels est sujette aux erreurs et rend le code plus difficile à lire.

[C++ Core indication pour cet avertissement](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#c128-virtual-functions-should-specify-exactly-one-of-virtual-override-or-final)
