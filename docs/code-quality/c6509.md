# <a name="c6509"></a>C6509
AVERTISSEMENT C6509 : annotation non valide : 'return’ne peut pas être référencé à partir d’une condition préalable

 Cet avertissement indique que le mot clé `return` ne peut pas être utilisé dans une condition préalable. Le mot clé `return` est utilisé pour mettre fin à l’exécution d’une fonction et retourner le contrôle à la fonction appelante.

## <a name="example"></a>Exemple
 Le code suivant génère cet avertissement, car `return` est utilisé dans une condition préalable :

```cpp
#include <sal.h>

int f (_In_reads_(return) char *pc)
{
    // code ...
    return 1;
}
```

 Pour corriger cet avertissement, utilisez le code suivant :

```cpp
#include <sal.h>

int f (_In_reads_(i) char *pc, int i)
{
    // code ...
    return 1;
}
```
