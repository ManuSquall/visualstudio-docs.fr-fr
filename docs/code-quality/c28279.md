# <a name="c28279"></a>C28279
AVERTISSEMENT C28279 : pour le symbole, un’BEGIN’a été trouvé sans’end’correspondant

 Le langage d’annotation prend en charge une construction Begin et end (`{` et `}` en C), et l’appariement a été déséquilibré. Cette situation peut être évitée si les macros sont utilisées.
