# <a name="c28204"></a>C28204
avertissement C28204 : \<fonction > : Seule de cette surcharge et celle de \<filename > (\<ligne >) sont annotées pour \<paramname > : les deux ou aucune des deux doivent être annotés.

 Cet avertissement est émis lorsqu’il existe une erreur dans les annotations.
