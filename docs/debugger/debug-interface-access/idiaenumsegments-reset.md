# <a name="idiaenumsegmentsreset"></a>IDiaEnumSegments::Reset
Réinitialise la séquence d’énumération au début.  
  
## <a name="syntax"></a>Syntaxe  
  
```C++  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>Valeur de retour  
 Retourne `S_OK`.
