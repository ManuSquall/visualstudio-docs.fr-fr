# <a name="idiaenumdebugstreamdatareset"></a>IDiaEnumDebugStreamData::Reset
Réinitialise le début d’une séquence d’énumération.  
  
## <a name="syntax"></a>Syntaxe  
  
```C++  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>Valeur de retour  
 Retourne `S_OK`.
